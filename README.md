# ai_coding_template
Template project with helpful defaults for starting new ai-coding projects.


## Usage

1. Clone the repository.
2. Start prompting.


## Agent Behavior

The [`AGENTS.md`](https://github.com/whogben/ai_coding_template/blob/main/AGENTS.md) shapes the AI's behavior:
- Records all design decisions in `DESIGN.md`.
- Maintains the README and CHANGELOG to match the current state.
- Pushes to git after successful changes.
